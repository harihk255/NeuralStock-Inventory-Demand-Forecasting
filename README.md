# NeuralStock

AI-Powered Inventory Demand Forecasting System built using Python, Streamlit, and PyTorch.

## Overview

NeuralStock is a deep learning-based inventory demand forecasting application that helps businesses predict future stock demand and improve inventory planning.

## Features

* Inventory demand forecasting using LSTM
* Interactive Streamlit dashboard
* Data visualization with Plotly
* Deep learning model built with PyTorch
* Demand trend analysis

## Technologies Used

* Python
* Streamlit
* PyTorch
* Pandas
* NumPy
* Plotly

## Project Structure

* app/ - Streamlit application
* data/ - Dataset files
* models/ - Trained model files
* notebooks/ - Development and experimentation notebooks
